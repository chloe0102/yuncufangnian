<div class="char-alias">軼聞</div>

  <dt>（一）</dt><dd>與已故宸元皇貴妃洛清紓為手帕交，對方曾是她的伴讀。</dd>
  <dt>（二）</dt><dd>熙箴二十七年整整一年不曾於大眾露面，僅在熙箴帝大喪時短暫出現，而後再見便是秦宣帝登基典，她已變得霸道蠻橫，不講道理。</dd>
  <dt>（三）</dt><dd>長公主府曾被周言杰餘黨放火燒盡，為修養身心，自那以後便居於宮內有梅宮。</dd>
  <dt>（四）</dt><dd>數年來因不明原因一再推拒世事，陰晴不定，行蹤飄忽，直至秦宣五年徹底放手，任誰來都端著一副清冷如水，唯獨秦宣帝貌似很惹她厭煩。</dd>
  <dt>（五）</dt><dd>看淡是非，平時總是一身清雅衣衫，對誰都冷淡的緊。</dd>
