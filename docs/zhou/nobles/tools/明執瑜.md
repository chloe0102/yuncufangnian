<div class="char-alias">軼聞</div>
<dl class="char-dl">

  <dt>（一）</dt>
  <dd>
  明氏世為朝中文臣重族，其父與兄長皆深受康璟帝周登昀信任。
  </dd>

  <dt>（二）</dt>
  <dd>
  康璟帝周登昀尚為燕靖王時，常出入明府與明懷信論策，明執瑜年少時亦曾遠遠見過數回，自此對其人頗為仰慕。
  </dd>

  <dt>（三）</dt>
  <dd>
  明家素有送女入宮之意，原先曾議以庶女明執嫻入侍秦宣帝周裴聿，然秦宣帝終未開選秀，此事遂止。
  </dd>

</dl>
