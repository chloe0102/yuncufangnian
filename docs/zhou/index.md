# 周國

<div class="world-intro">

周國，位於中原大陸東北方，幅員遼闊，地勢豐饒而多變。  
西北接沙漠綠洲，良駒與信鴿皆出於此；東北濱海，握有中原大陸唯一對外港口，航運昌盛，為眾國往來交流之樞紐。  
今與樓蘭結盟，庇護殷國，並與戚國、波斯交惡。

</div>

## 州府地誌

### 京畿與中樞
- **昌黎**：首都京城，皇宮所在之地。**（總督）**
- **鄴州**：京城之北，盛產羊脂白玉。**（知州）**
- **虞城**：京城之南，盛產樂子伶人。**（知府）**

### 東境與海港
- **敷城**：京城東北，盛產沉水南珠。**（知府）**
- **臨潢**：京城之東，港阜所在之地。**（總督／同知）**
- **臨安**：京城東南，盛產黃潤柏木。**（知縣）**

### 西境與山漠
- **平涼**：京城西北，沙漠平原交界。**（知縣）**
- **天山**：京城之西，高山巍峨聳立。**（知縣）**
- **永平**：京城西南，盛產千里信鴿。**（知府）**

### 邊境重地
- **懷遠**：接壤樓蘭，盛產伏犁汗馬。**（總督／同知）**
- **和州**：接壤殷國，盛產翎山寒茶。**（知州）**
- **豫州**：接壤戚國，盛產珍稀藥材。**（知州）**
- **廣寧**：接壤波斯，盛產紅髓玉石。**（總督／同知）**
