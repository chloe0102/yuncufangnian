# 定元史

### 定元元年

<div class="history-item">
周子信冊髮妻李氏為元德皇后。后性端淑，早歲從帝於軍旅之中，艱難備嘗，軍中皆敬之。其嫡長女冊為襄陽公主。
</div>

<div class="history-item">
周子信念從征諸將功最著者，乃分封六地侯爵，以守四方：秦氏封於永平、白氏封於廣寧，以備波斯；李氏居虞城，食沃土，以為外戚；步氏鎮豫州，扼南境要衝；侯氏守和州，控東南關隘；許氏封於懷遠，總西道商路與邊防。自是六家世為武門，分鎮邊疆。
</div>

<div class="history-item">
董氏為幕府軍師，北征之初，諸軍進退、糧道分配、諸侯攻伐，多出其議，冊為左相；薛氏善辯而敏達，長於交涉，北地諸侯未定之時，數往諸國說降，使多國不戰而附，冊為右相；羅氏出寒門而學識淵博，帳下文書、軍令多由其整理，冊為內閣學士；趙氏亦為幕府謀士，其策多主權變，善因勢制宜，授中書侍郎。
</div>
