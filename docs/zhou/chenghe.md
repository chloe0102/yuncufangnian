# 盛和史

### 盛和元年

<div class="history-item">
周珵冊太子妃薛氏為文恭皇后。
</div>

### 盛和七年

<div class="history-item">
天下承平，然武門世族勢重。六侯鎮邊既久，部曲甚眾，朝廷多所忌。周珵以邊境已定，軍制當歸於朝廷，乃議整軍政，又詔開國諸侯，仍保其爵祿，然自此不復世襲。
</div>

### 盛和十年

<div class="history-item">
秦氏、白氏世鎮西南，邊功最著，軍紀嚴整，帝以其守邊之勞，特許仍持兵鎮舊地；其餘諸侯，多徙居昌黎，步氏其子弟兵編入禁軍，虎符未歸；侯氏撤邊鎮、許氏邊軍裁撤，歸還虎符，改任內官；李氏以外戚之故，仍享封邑，而不復領軍。
</div>
