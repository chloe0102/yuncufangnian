# 殷國

<div class="world-intro">

殷國位於中原大陸東南方，疆域為五國之中最小，卻地靈人傑。  
江南水土溫潤，平原廣袤，河湖縱橫，向以農桑聞名。  
境內稻作與養蠶並盛，所產絲綢細潤柔麗，為中原貴族與宮廷衣料之上品。

殷國歷代帝王性情溫和，不好權爭，國力亦因而漸衰。  
然其地處交通與水路要衝，為兵家必爭之地，故長年受周國庇護而存。

## 州府地誌

### 京畿之地
- **汴京**：殷國首都，皇宮所在之地。朝堂百官雲集，文教興盛，市肆繁華。

### 北境軍門
- **汴北**：接壤周國，為殷國北門關。軍屯要地，土質鬆潤，氣候乾爽，盛產白菘。

### 江南稻鄉
- **汴南**：京南水網縱橫，稻米豐饒，亦多蠶桑。洞庭湖上游水源由此流入，亦為水患前沿之地。

### 湖澤水鄉
- **太湖**：洞庭湖所在之地，丘野與湖泊交錯，水路曲折。盛產碧螺春與黃茶。
- **湘西**：洞庭湖下游，民多舟居，水患頻仍。多以漁牧、種藕為生，亦善草編與織補。

### 農茶之州
- **望江**：接壤戚國，江水交會之地。土壤肥沃，茶園密布，盛產雨前龍井。

### 絲綢之鄉
- **姑蘇**：蠶桑繁盛之地，織戶聚居。刺繡與染坊並盛，為殷國絲綢主產州。

### 樂藝之地
- **饒陽**：接壤戚國，地近京畿。樂坊與繡肆林立，盛產歌伎樂工與巧匠文士。
